## SCORING GUIDE

| Score | Grade | Assessment |
|-------|-------|------------|
| 270-300 | A+ | Excellent - Minor optimizations only |
| 240-269 | A | Great - Few improvements needed |
| 210-239 | B | Good - Some gaps to address |
| 180-209 | C | Average - Notable improvements needed |
| 150-179 | D | Below Average - Significant work needed |
| 0-149 | F | Poor - Major overhaul required |

## QUICK AUDIT COMMANDS

### Check robots.txt
```bash
curl -s https://example.com/robots.txt
```

### Check sitemap
```bash
curl -s https://example.com/sitemap.xml | head -50
```

### Check page speed (requires API or manual)
```
https://pagespeed.web.dev/report?url=https://example.com
```

### Check schema
```
https://validator.schema.org/
```

## OUTPUT FORMAT

For each site, produce:
1. **Score Summary** - Total points per section
2. **Top 5 Issues** - Most critical fixes needed
3. **Top 5 Quick Wins** - Easy improvements
4. **Detailed Breakdown** - Full scoring table
