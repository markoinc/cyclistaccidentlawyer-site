## Key Themes Across Sources

### 1. Simplification is Key
- 2-campaign structures beat complex setups
- Broad targeting often outperforms detailed
- Let AI/algorithms do more of the work

### 2. Creative is the New Targeting
- UGC dominates performance
- Authenticity beats polish
- Short-form video (Reels, Stories) essential
- Weekly creative testing recommended

### 3. Data Quality Matters More Than Ever
- Clean conversion tracking is foundational
- Server-side tracking (CAPI) recommended
- First-party data increasingly valuable

### 4. AI/Automation Adoption
- Advantage+ audiences and campaigns
- CBO for budget optimization
- Dynamic creatives for personalization
- Let learning phase complete before changes

### 5. Budget Strategy
- Need sufficient budget for learning phase (~50 conversions/week)
- CBO for scaling, ABO for testing
- 20% budget increase rule for stability
- 70/30 split between scaling and testing
