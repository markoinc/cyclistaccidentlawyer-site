# SMS Follow-Up Sequences - Templates & Strategies

**Source:** Industry best practices from Omnisend, TextDrip, CleverTap  
**Topic:** Follow-up text message sequences for different scenarios  
**Last Updated:** January 2026

---

## Overview

Follow-up is where deals are won or lost. 80% of sales require 5+ follow-ups, yet most salespeople stop after 1-2. SMS follow-up sequences automate persistence while maintaining a personal touch.

---

## Core Principles

### The Follow-Up Rules

1. **Provide Value Each Time** - Don't just "check in"
2. **Vary Your Approach** - Different angles, same goal
3. **Respect Timing** - Space messages appropriately
4. **Make It Easy to Reply** - Clear CTAs
5. **Know When to Stop** - Respect opt-outs and silence

### Optimal Follow-Up Timing

| Touch | Timing | Purpose |
|-------|--------|---------|
| 1 | Immediate | Initial response |
| 2 | 2-4 hours | If no reply, gentle nudge |
| 3 | 24 hours | Add value, new angle |
| 4 | 3 days | Social proof/case study |
| 5 | 7 days | Urgency or new offer |
| 6 | 14 days | Last attempt / breakup |

## Sequence 1: New Lead Follow-Up

### Trigger: Form Submission / Lead Capture

**Message 1 (Immediate):**
```
Hey {{first_name}}! This is {{rep_name}} from [Business].

Just saw your [form submission/download/request]. 
What questions can I answer for you?
```

**Message 2 (4 hours, if no reply):**
```
{{first_name}}, following up! I know you're probably busy.

Quick question: What's your biggest challenge with 
[topic they expressed interest in]?

Would love to help if I can.
```

**Message 3 (24 hours):**
```
Hey {{first_name}}, thought you might find this useful:

[Brief tip or insight related to their interest]

Want me to share how we help clients with this?
```

**Message 4 (3 days):**
```
"We went from [problem] to [result] in just [timeframe]."

That's what [Client Name] said about working with us.

Curious if similar results would help you, {{first_name}}?
```

**Message 5 (7 days):**
```
{{first_name}}, I have a few minutes open [tomorrow/this week] 
if you're ready to chat about [their goal/problem].

Reply with a time that works, or let me know if 
the timing isn't right.
```

**Message 6 (14 days - Breakup):**
```
Hey {{first_name}}, I don't want to keep bugging you!

Seems like now might not be the right time, and that's 
totally okay.

If things change, I'm here. Just reply to this text anytime.

Best,
{{rep_name}}
```

## Sequence 6: Re-Engagement (Cold Leads)

### Trigger: No Response in 30+ Days

**Message 1:**
```
Hey {{first_name}}, it's been a while!

You reached out about [topic] back in [month].
Is that still something you're working on?

Reply YES if you'd like to reconnect.
```

**Message 2 (7 days if replied YES):**
```
Great to hear from you, {{first_name}}!

Things have actually evolved since we last talked.
We now [new feature/capability/offer].

Worth a quick chat to catch you up?
```

**Message 2 (7 days if no reply):**
```
{{first_name}}, no worries if things have changed!

Just in case: We just released [new thing] that's 
getting great results for [similar companies/people].

Reply INFO if you want details.
```

**Message 3 (14 days - Final):**
```
Last one from me, {{first_name}}!

If [solving problem/achieving goal] is ever 
a priority again, just text me.

I'm here when you're ready!
{{rep_name}}
```

## Follow-Up Message Frameworks

### The Value-Add Framework

```
[Personalized opener]

[New piece of value - tip, insight, resource]

[Soft CTA that's easy to respond to]
```

**Example:**
```
Hey {{first_name}}, quick tip:

Most of our successful clients [do specific thing].
It usually [achieves specific result].

Want me to show you how?
```

### The Social Proof Framework

```
[Reference to their situation]

[Similar client's result]

[Implied "you could have this too" CTA]
```

**Example:**
```
{{first_name}}, this reminded me of you:

[Client Name] was in a similar spot - [situation].
They're now [impressive result].

Curious if we could do the same for you?
```

### The Urgency Framework

```
[Time-sensitive element]

[What they'll miss]

[Easy action to take]
```

**Example:**
```
Heads up, {{first_name}} - [limited availability/deadline].

After [date], [what changes].

Want to lock in [benefit] before then?
```

### The Breakup Framework

```
[Acknowledge you've been persistent]

[Give them an easy out]

[Leave door open]
```

**Example:**
```
{{first_name}}, I don't want to be "that" salesperson.

Seems like now isn't the right time, and I respect that.

If things change, just text me. I'll be here!
```

## Pro Tips

### 1. Track What Works
- A/B test different messages
- Note which touchpoint gets the most replies
- Double down on winning templates

### 2. Personalize Beyond the Name
- Reference their specific interest
- Mention their industry/company
- Recall previous conversation points

### 3. Vary Message Length
- Mix short punchy texts with longer value-adds
- Short for follow-ups: "Thoughts?"
- Longer for value: Insights + CTA

### 4. Use Triggers, Not Just Time
- Trigger follow-up when they open your email
- Follow up when they visit your website
- Reach out when they engage on social

### 5. Know When to Stop
- After 6-8 touches with no response, pause
- Re-engage in 30-60 days with new angle
- Respect explicit "not interested" responses

## Summary

**The Follow-Up Formula:**
1. **Immediate:** Respond fast, be personal
2. **Value-First:** Each message should offer something
3. **Variety:** Different angles, same goal
4. **Persistence:** Most deals need 5+ touches
5. **Respect:** Know when to pause or stop
6. **Automate:** But keep it human

**Remember:** The fortune is in the follow-up. Don't give up after one message.
